イベント編

イベントとは
HTMLやCSSを変化や変更させるための何らかのきっかけの事。
$('document').ready(function( ){
 HTMLドキュメントが読み込まれたら処理をする
});
今までやってきたjQueryも上記のイベント(ドキュメントを読み込んだら処理をしなさい)が前提となっている。 これを省略して書いたものが以下で
$(function( ){
 HTMLドキュメントが読み込まれたら処理をする
});
上記のように書き換えることができ、これを書かないとjQueryは正しく動かない。それはドキュメントのページを読み込み終えていないからで
jQueryは最初に$(function)( ){ }を書き、{…}の内側に
スクリプトを書いていく。
イベントの書き方
$('セレクター')
.イベント名
(function( ){
 指定されたイベントが実行する処理
});

mousedown/mouseupイベント

$('セレクター')
.mousedown
(function(){
 マウスダウンされた時に実行する処理
});
$('セレクター')
.mouseup
(function( ){
 マウスアップされた時に実行する処理
});
$('#btn').mousedown(function( ){

 $('img').attr('src','image/ヒョウ１.jpg').attr('atr','ヒョウ１');

});
$('#btn').mouseup(function( ){

 $('img').attr('src','image/ぬこ５.jpg').attr('atr','ぬこ５');;

});

hover( )

hoverイベント
対象にカーソルが乗った時や降りた時に処理が実行される点ではmouseover( )/mouseout( )と似ているがhover( )の場合は指定された要素のみに影響を与える。
hoverイベントは乗った時と降りた時の処理を両方書く必要がある
$('セレクター')
.hover
(function( ){
 マウスが重なった時に実行する処理
  },function( ){
 マウスが外れた時に実行する処理
});
ここまでおいで
$('.box2').hover(function( ){
白いboxにマウスカーソルなどが重なると
  $('this').animate({fontSize:'24px'}, 500);
boxの中の文字サイズが大きくなる
  },function( ){
白いboxからマウスカーソルが外れると
  $('this').animate({fontSize:'16px'}, 500);
boxの中の文字サイズが戻る
});

特別な理由がなければhover( )を使う方が良い

mousemove( )

mousemoveイベント
特定の要素上でカーソルが動いた場合に処理を実行する。主に座標情報を取得するのに使われる。
function( )を(e)にすることでイベント発生時にe.clientXでX軸の座標、e.clientYでY軸の座標を取得できる。
$('セレクター')
.mousemove
(function( ){
 設定した要素上でマウスが動いている時に実行する処理
});
この中でマウスを動かすと座標が取得できる
box内でのX座標
box内でのY座標
$('.box').mousemove(function(e){
box内でマウスが動くと
 $('#inputX').val(e.offsetX);
inputXにX軸の座標が書かれる
  $('#inputY').val(e.offsetY);
inputYにY軸の座標が書かれる
});

eの後ろのプロパティを変えることで取得できる座標が変わる。上の例だとoffsetを設定しているので指定した要素内の座標を取得し、clientにするとセレクターを基準に表示画面内の座標を取得できる。例はセレクターをHTMLとしているので画面の表示領域すべてを取得できる。
表示領域のX座標
表示領域のY座標
$('html').mousemove(function(e){
表示画面でマウスが動くと
 $('#inputX').val(e.clientX);
inputXにX軸の座標が書かれる
  $('#inputY').val(e.clientY);
inputYにY軸の座標が書かれる
});

one( )

oneイベント
指定したイベントを一回だけ実行する。.oneを書いた後に( )内にイベントの発生条件を書き,(カンマ)で区切ってfunction( )以降を書いていく。イベントの発生条件にはclick,dblclick,mousedownなどが入る
$('セレクター')
.one('イベントの発生条件',
function(){
 イベント発生時に実行する処理
});

画像にカーソルが乗ると１回目は変わるが２回目は変わらない
$('#cat').one('mouseover',function( ){
画像にカーソルが乗ると
 $(this).attr('src','image/ぬこ１２.jpg').attr('alt','ぬこ１２');
img要素のsrc属性とalt属性をattr( )で変える
 }).one('mouseout',function( ){
画像からカーソルが降りると
 $(this).attr('src','image/ぬこ１１.jpg').attr('alt','ぬこ１１');
画像が戻る
});

クリックイベントを一回押すと画像が変わるが二回目にクリックすると画像のリンク先に飛ぶ

 イベント発生時に実行する処理
