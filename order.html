メソッド編

メソッドとは
セレクターで指定した要素を操作する命令。
$(セレクター)
.命令
( );
セレクターを指定して命令を設定

テキストの変更と取得

テキストの変更
.text( )を使うことでテキストの変更ができる。テキストは” ”(ダブルクォーテーション)または' '(シングルクォーテーション)で囲う。
$('セレクター').text(" ");

<h3>変更前</h3>

$('h1').text('changed!');
h1要素のtextを'change!'に書き変える
テキストの取得
( )内を空欄にして取得する。また取得した文字列をそのまま入れることもできる。
$('セレクター').text( );
<h3>取得されるもの</h3>

<p>変更前</p>

$('p').text($('h3').text( ));
p要素にh3のテキストを取得して入れる

HTMLの変更と取得

HTMLの変更
( )内に変更したいHTMLを入力する。textと同じ様に” ”(ダブルクォーテーション)または' '(シングルクォーテーション)で囲う。
$('セレクター').html(" ");

<p>変更前</p>

$('p').html("<strong変更後</strong">);
p要素を<strong>変更後</strong>に変える
HTMLの取得
( )内を空欄にして取得する。こちらも取得したhtmlをそのまま入れることができる。
$('セレクター').html( );
変更前

<p>変更前</p>
$('p').html($('#getHtml').html());
id="getHtml"の値を取得してp要素に書き出す

HTMLの挿入と移動

HTMLの挿入
  • $('セレクター').prepend( );
    • 要素内の指定した子要素の先頭に要素を挿入する
  • $('セレクター').append( );
    • 要素内の指定した子要素の最後に要素を挿入する
  • $('セレクター').before( );
    • 指定した要素の前に挿入する
  • $('セレクター').after( );
    • 指定した要素の後ろに挿入する
  • list0
  • list1
  • list2
  • list3
$('ul>li:eq(0)').prepend('<strong>prepend</strong>');
ul直下にある1番目のli要素の先頭にstrong要素を挿入する
$('ul>li:eq(3)').append('<strong>append</strong>');
ul直下にある4番目のli要素の最後にstrong要素を挿入する
$('ul>li:eq(0)').before('<h3>before</h3>');
ul直下にある1番目のli要素の前に挿入する
$('ul>li:eq(3)').after('<h3>after</h3>');
ul直下にある4番目のli要素の後ろに挿入する
「(場所)に(挿入するもの)を…」の順番
新しい要素を作って挿入ができる
HTMLの移動

    • 要素内の指定した子要素の先頭に要素を挿入する
  • $('セレクター').appendTo( );
    • 要素内の指定した子要素の最後に要素を挿入する
  • $('セレクター').insertBefore( );
    • 指定した要素の前に挿入する
  • $('セレクター').insertAfter( );
    • 指定した要素の後ろに挿入する
  • list0
  • list1

  • list3

prependTo appendTo

insertBeforeで移動させるもの

insertAfterで移動させるもの

$('strong').prependTo('ul>li:eq(0)')
strong要素をul直下にある1番目のli要素内の先頭に挿入する
$('strong').appendTo('ul>li:eq(3)')
strong要素をul直下にある4番目のli要素内の最後に挿入する
$('h3').insertBefore('ul>li:eq(0)')

$('h3').insertAfter('ul>li:eq(3)')

「(挿入するもの)を(場所)に…」の順番

属性値

属性値の変更
属性値を変更できる
$('セレクター').attr('属性名','属性値');
セレクターの属性名を指定し値を変更する
googleへ
$('a').attr('href','https://www.yahoo.co.jp/');
a要素のhref属性を'https://www.yahoo.co.jp/'に変えている
属性値の取得
セレクターを指定して取得する。また取得した値をそのまま入れることもできる
$('セレクター').attr(" ");
セレクターの属性を取得する
googleへ

$('p').text($('a').attr('href'));
a要素のhref属性を取得してp要素に書き出す

class属性

class属性の追加
addClass( )を使う事でclassを追加できる。
$('セレクター').addClass('class名');
class属性の削除
removeClass( )を使う事でclassを削除できる。
$('セレクター').removeClass('class名');

<p>こんにちは</p>

$('p').addClass('attrClass');
p要素にクラスを追加
$('p').removeClass('attrClass');
p要素からクラスを削除
hasClass( )
( )内にそのクラスがあるか無いかを判定する。あればtrueを返し、無ければfalseを返す。
$('セレクター').hasClass('class名');

<p>どっちでしょう</p>

if($('p').hasClass('judge')){
p要素が'judge'というclassを持っていれば
  $('p').text('持ってます!');
p要素のテキストを「持ってます！」に変える
 }else{
そうでなければ
  $('p').text('持ってません');
p要素のテキストを「持ってません！」に変える!
  }
});
toggleClass( )
classの付け外しができる。
$('セレクター').toggleClass('class名');

<p>クラスの付け外し</p>

$('p').toggleClass('toggle');
p要素にtoggleclassを付け外しできる様にする

CSSプロパティ

CSSプロパティの設定
css("プロパティ名","値")を使うとプロパティの値を変えたり取得できたりする。
$('セレクター').css("プロパティ","値");

<h3>テキストテキストテキストテキストテキスト</h3>

$('h3').css("color","red");
h3要素のcss(文字色を赤に)を変更
複数のプロパティを変える場合
プロパティ名と値の間を : ( コロン )で区切り値を" "(ダブルクォーテーション)で囲い , ( カンマ )で繋げる。
$('セレクター').css({
 "プロパティ名" : "値",
 "プロパティ名" : "値"
});

<h3>テキストテキストテキストテキストテキスト</h3>

$('#change-css').css({
  "color":"red",
  "backgroundColor":"yellow",
});
h3要素のcss(文字色を赤、背景色を黄色に)を変更
CSSプロパティの取得
テキストやhtmlのようにcss("プロパティ名")とすると取得できる。
$('セレクター').css("プロパティ名");

<h3>テキストテキストテキストテキストテキスト</h3>

$('p').text($('h3').css("color"));
h3要素のcss(文字色)を取得してp要素に記述

scroll( )
topからの距離を指定してそこへスクロールさせることができる。また( )内に数字を入れることで任意の場所へスクロールできる。
$('html,body')とセットで使う
$('html,body').scrollTop( );
$('html,body').scrollTop(0);
ページのtopから0のところに移動
offset( )
$('セレクター').offset( );
$('セレクター').offset( ).top;
要素の表示位置を取得できる。また後ろに.topとすることでtopからの位置を取得できる。
