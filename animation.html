アニメーション編

アニメーションとは
セレクターを指定して命令を出すことで動きをつけられる。
$(セレクター)
.アニメーション
( );

イベントと併せて使われることが多い。
$('セレクター').イベント(function( ){
 $('セレクター').アニメーション( );
});

show( )/hide( )

show( );/hide( );
show( )は表示、hide( )は非表示にする。
$('セレクター')
.show
(スピード);
$('セレクター')
.hide
(スピード);
( )内には「"fast"」,「"slow"」のいずれか、またはミリ秒単位で(1000ミリ=1秒)スピードを入力する。ミリ秒で指定する場合は” ”で囲わなくても良い。何も記入しないと即座に変化する。
$('#showBtn').click(function(){
表示ボタンを押すと
 $('#box').show("slow");
boxが現れる
});
$('#hideBtn').click(function(){
非表示ボタンを押すと
 $('#box').hide("fast");
boxが消える
});
表示の方は「"slow"」非表示の方は「"fast"」を設定した。
toggle( );
一つのコマンドで表示、非表示をする。
$("セレクター")
.toggle
(スピード);
$('#btn').click(function(){
ボタンを押すと
 $('#box').toggle(500);
boxが現れたり、消えたりする
});
クリックイベントの場合ボタンを連打するとその数だけアニメーションが実行されてしまう。
:not(:animated)
と組み合わせることで解決できる。これを書くとアニメーションが処理中の時は処理が実行されない。
$('#btn').click(function(){
ボタンを押すと
 $('#box:not(:animated)').toggle(500);
boxが現れたり、消えたりする
});

slideDown( )/slideUp( )

slideDown( )/slideUp( )
slideDown( )はセレクターで指定した要素をスライドダウンで表示、slideUp( )はスライドアップで非表示をすることができる。
$('セレクター')
.slideDown
(スピード);
$('セレクター')
.slideUp
(スピード);
$('#slidedownBtn').click(function( ){
表示ボタンを押すと
 $('#box').slideDown('slow');
boxが現れる
});
$('#slideupBtn').click(function( ){
非表示ボタンを押すと
 $('#box').slideUp('fast');
boxが消える
});
slideToggle( )
slideDown( )/slideUp( )にもtoggleが使える。
$('セレクター')
.slideToggle
(スピード);
$('#btn').click(function( ){
ボタンを押すと
 $('#box:not(:animated)').slideToggle("slow");
boxが現れたり、消えたりする
});

fadeIn( )/fadeOut( )

fadeIn( )/fadeOut( )
フェードインしながら要素を表示、非表示する。
$('セレクター')
.fadeIn
(スピード);
$('セレクター')
.fadeOut
(スピード);
$('#fadeinBtn').click(function(){
表示ボタンを押すと
 $('#box').fadeIn("slow");
boxが現れる
 });
$('#fadeoutBtn').click(function(){
非表示ボタンを押すと

boxが消える

fadeTo( )
セレクターで指定した要素の透明度を設定まで徐々に変更する。
$('セレクター')
.fadeTo
("スピード",
透明度
);
$('#btn').click(function(){
ボタンを押すと
 $('#box').fadeTo("slow",0.5);
boxが半透明(opacity:0.5みたいな感じ)になる
});
fadeToggle( )
fadeIn( )/fadeOut( )にもtoggleが使える。
$('セレクター')
.fadeToggle
(スピード);
$('#btn').click(function(){
ボタンを押すと
 $('#box':not(:animated)').fadeToggle("slow");
boxが現れたり、消えたりする
 });
fadeOut(" ")とfadeTo(" ",0)は似ているが結果に違いがある。fadeOut(" ")の方はスペースを埋めてしまうのに対して、fadeTo(" ",0)は透明度が０になっているだけで実際はそこに存在している。使う時はどちらの結果を得たいかで使い分ける。
またshow( )/hide( )シュッとfadeはフワッとというイメージで使い分けるのが良い。

animate( )

animate( )
animateで設定できるのは数値であらわせるプロパティのみとなる(bakground-colorやdisplayはできない)。
$("セレクター")
.animate
({
  "cssセレクタ": "css数値"
 });
これだけで最低限書ける
$('#btn').click(function( ){
ボタンを押すと
 $('#box').animate({
boxのアニメーションを
  "margin-left": "500px",
500px左に移動する(余白を空ける)
 });
});
{ }の後ろをカンマで区切って時間をミリ秒単位で指定する事でアニメーションのスピードを変えられる。

 },
アニメーション時間
);

500px左に移動する

1000
